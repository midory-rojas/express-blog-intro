# Express-blog-intro

# Obbiettivo:

Creiamo il nostro blog personale e giorno dopo giorno lo potremo arricchire con nuove funzionalità sulla base di quello che impareremo. 

# Dipendenze:

- Express
- Testare su postman

# Esercizio:

- [x]Creiamo il progetto base con una rotta / che ritorna un testo semplice con scritto ”Server del mio blog”
- [x]Creiamo un array dove inserire una lista di almeno 5 post, per ognuno indicare titolo, contenuto, immagine e tags (tags è un array di stringhe)
- [x]Creiamo poi una rotta /bacheca che restituisca un oggetto json con la lista dei post e il conteggio, partendo da un array.
- [x]Configuriamo gli asset statici sull’applicazione in modo che si possano visualizzare le immagini associate ad ogni post.
